# mh_Astro_Weather_Widget (v1.1.0.0)

A compact, always-on-top Weather + Astro widget with a night-vision friendly startup shield.

## Quick Start (ZIP Release)

1. Download: **mh_Astro_Weather_Widget.zip**
2. Extract it anywhere.
3. Run: **mh_Astro_Weather_Widget_Launcher.exe**

> **Black / blank screen at launch?**  
> That is intentional: a fullscreen shield briefly covers the screen during startup to prevent bright Windows flashes, preserve night vision, and avoid flicker.

## Key Features

- Auto location (IP) fallback + manual location entry (lat/lon, city label)
- Temperature, sky condition, dew point, dew-risk, pressure, wind
- **Sunrise / Sunset**
- **Moon phase + Moonrise / Moonset**
- System tray: Show / Hide, Refresh, Settings, Quit

## Running From Source (optional)

If running the `.py` script directly, install dependencies:

- `pip install PyQt6 requests astral`
